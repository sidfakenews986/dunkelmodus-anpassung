# Beitrag zum Projekt

Vielen Dank, dass Sie erwägen, zu diesem Projekt beizutragen! Hier sind einige Schritte, um Ihnen den Einstieg zu erleichtern:

1. Fork dieses Repository.
2. Erstellen Sie einen neuen Branch (`git checkout -b feature/neue-funktion`).
3. Nehmen Sie Ihre Änderungen vor und testen Sie diese.
4. Fügen Sie Ihre Änderungen hinzu (`git add .`).
5. Commiten Sie Ihre Änderungen (`git commit -m 'Fügen Sie neue Funktionen hinzu oder beheben Sie Fehler'`).
6. Pushen Sie den Branch (`git push origin feature/neue-funktion`).
7. Erstellen Sie einen Pull-Request.

Bitte stellen Sie sicher, dass Ihr Code den bestehenden Standards und Stilrichtlinien folgt.

Viel Spaß beim Programmieren!
