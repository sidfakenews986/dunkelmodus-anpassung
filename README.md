# Dunkelmodus-Anpassung

Willkommen zu Dunkelmodus-Anpassung! Diese Bibliothek hilft Ihnen, den Dunkelmodus in Ihrer Webanwendung schnell und unkompliziert zu implementieren und anzupassen. 

## Funktionen
- Einfache Aktivierung des Dunkelmodus mit nur wenigen Zeilen Code.
- Anpassbare Farbpaletten, die auf Ihr Design abgestimmt sind.
- Unterstützung für CSS-Variablen, sodass Sie die Farben flexibel gestalten können.
- Möglichkeit zur Speicherung der Benutzereinstellungen im lokalen Speicher.

## Installation
```bash
npm install dunkelmodus-anpassung
```

## Nutzung
```javascript
import { activateDarkMode } from 'dunkelmodus-anpassung';

activateDarkMode();
```
